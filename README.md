# Из Author.Today в нейродетектор Яндекс.

На сайте [author.today](https://author.today) запрещено копировать текст, выделяя мышкой или с клавиатуры. Но копировать текст нужно для отправки в [инструмент проверки](https://yandex.ru/lab/neurodetector) -- написано это человеком или нейросетью. Данное расширение для Chrome автоматизирует этот процесс:

- извлекает текст текущей главы из `#text-container`;
- копирует текст в буфер обмена;
- отправляет текст в [Нейродетектор Яндекса](https://yandex.ru/lab/neurodetector);
- показывает вероятность того, что текст сгенерирован нейросетью.

Репозиторий: [https://github.com/pakuula/at-chapter-to-yn-detector](https://github.com/pakuula/at-chapter-to-yn-detector)

## Установка из ZIP релиза на GitHub

1. Откройте страницу релизов:
   [https://github.com/pakuula/at-chapter-to-yn-detector/releases](https://github.com/pakuula/at-chapter-to-yn-detector/releases)
2. Скачайте файл `at-chapter-to-yn-detector-extension.zip` из последнего релиза.
3. Распакуйте ZIP в отдельную папку, например `C:\Extensions\at-chapter-to-yn-detector`.
4. Откройте в Chrome страницу `chrome://extensions/`.
5. Включите переключатель **Режим разработчика** (в правом верхнем углу).
6. Нажмите **Загрузить распакованное расширение**.
7. Выберите папку, в которую вы распаковали ZIP (папку с `manifest.json`).

После этого расширение появится в списке и будет готово к работе.

## Как использовать

1. Откройте страницу главы на author.today.
2. Нажмите на иконку расширения.
3. Выберите действие:
   - **Скопировать главу** — копирует текст главы в буфер; скопированный текст можно вручную отправить в Нейродетектор Яндекса;
   - **Проверить в нейродетекторе Яндекс** — отправляет текст в API Яндекса и показывает результат. Для отправки текста из расширения копировать текст не нужно, расширение само извлечет его и отправить в Нейродетектор.
4. Клик по уведомлению с результатом копирует результаты проверки в формате JSON в буфер обмена.

## Примечание

ZIP-файл из релиза устанавливается в Chrome как **распакованное** расширение (после распаковки), а не прямой установкой ZIP-файла.
